# Arachnid

## Setting Up

Install Front-End:
- `cd js`
- `npm install` to install dependencies

Install Engine
`cd engine`
`pip install -r requirements.txt`

Install Server:
- Stay in root directory
- `pip install -r requirements.txt`

Run Front-End:
- `cd js`
- `npm start`

Run Server:
- Stay in root
- `python server.py`


## Key Files

- `src/components/DataPane.js` is where vega loads the visualization and handles all the interactions
